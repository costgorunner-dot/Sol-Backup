# Research Quick Reference Card

**Before ANY research task, read this:**

## ✅ Must Do:
1. Use multiple search tools (Brave + Tavily)
2. Check minimum 3-5 sources
3. Find real user experiences (Reddit, GitHub)
4. Cross-reference all claims
5. Be transparent about methodology

## ❌ Never Do:
1. Trust first search result
2. Use only one source
3. Assume top 5 = best
4. Skip verification
5. Hide uncertainty

## 🔍 For Pricing:
- Check manufacturer + 3+ retailers + used markets
- Minimum 5 sources

## 🎯 For Performance:
- Real users (Reddit/GitHub) + benchmarks + official docs
- Verify model/workflow matches

## 📊 Always Include:
- Methodology used
- Sources checked
- Confidence level
- Caveats/uncertainty

**Remember:** Top results ≠ Best results. Dig deeper. Cross-reference. Verify.
